# Learning management system - Shared library

Shared library for Spring Boot RESTful microservices in Learning management system

[![](https://jitpack.io/v/medelafia/learning-management-system-shared-library.svg)](https://jitpack.io/#medelafia/learning-management-system-shared-library)

## Using with Maven

To use it in your Maven build add:
```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```

and the dependency:

```xml
<dependency>
    <groupId>com.github.medelafia</groupId>
    <artifactId>learning-management-system-shared-library</artifactId>
    <version>Tag</version>
</dependency>
```
